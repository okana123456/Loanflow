Quick Client Registration

  1. This opens IntaSend registration page in new tab
  2. You register the client's business
  3. IntaSend sends verification email to client
  4. Client verifies account (upload ID, KRA)
  5. You get API keys from IntaSend dashboard
  6. Come back here and paste API keys below
Get from: IntaSend Dashboard → API Keys
🔒 Kept secure - never shared with cashiers
Start with Test Mode to try it out, then switch to Live
💰 Cost: 3.5% + Ksh 10 per transaction
Example: Ksh 1,500 sale = Ksh 62.50 fee
✅ Benefits:
• No SMS phone needed at POS
• Automatic payment verification
• STK Push to customer phone
• Secure - cashiers never see payment details
• Falls back to manual if API fails
📋 Setup Instructions:
  1. Go to intasend.com and create account
  2. Verify your business (KRA PIN, ID, etc.)
  3. Get API keys from Dashboard → API Keys
  4. Enter keys above and enable integration
  5. Test with small transaction first

% of selling price
Sellers cannot sell any product below this % of its listed price. Leave empty or 0 for no restriction.
Example: 80 means a Ksh 100 item cannot be sold below Ksh 80.

Example: Set 7 AM → a sale at 6:30 AM counts as yesterday's business.
